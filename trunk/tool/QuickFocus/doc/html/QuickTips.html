Quick Tips

  1. You can test part of generated snip code by selecting a segement of code, then click on the "Run Snippet"  button in the inspection panel.
  2. You can create shortcut to current page loaded in QuickFocus by drag-and-drop the application icon to the Windows desktop. The application icon is located at the top left corner of QuickFocus's main window.
  3. You can undo changes made manually to the code snippet window by pressing the Ctrl-Z key.
