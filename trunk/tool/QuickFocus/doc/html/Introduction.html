Introduction

With the growing popularity of Internet web based applications have become a popular way to deliver information and services. Many tools have been developed to facilitate the test of web based applications. Among them the open source test framework IeUnit  provides a very simple yet efficient solution. With IeUnit users can write short scripts to control Internet Explorer to performed testing tasks.

However, IeUnit has the main difficulty that it not only requires good knowledge about the DHTML model, but  also good knowledge about the DOM structure of the page under test. The latter can often only be obtained by examining the page's HTML source. For large complex web pages direct examination without specially designed tools quickly become a tedious and error prone task.

QuickFocus is an application designed to help user overcome these difficulties. In particular, QuickFocus offers the following services:

  • Enable users to inspect HTML page by clicking directly on the rendered HTML page.
  • Suggest test code snippets to validate or control selected elements.
  • Execute test code snippets.
  • Generate simple templates for IeUnit test cases and IeUnit smart bookmarks.

With these features QuickFocus not only helps experienced programmer to develop testing scripts, but also novice users to quickly learn IeUnit framework and accomplish browser automation tasks.
