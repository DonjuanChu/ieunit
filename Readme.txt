Introduction:

This directory contains all source for the web testing 
framework IeUnit.


System Requirements:

    * Windows 2000 or higher.
    * Windows JavaScript engine v5.6 or higher
    * .Net Runtime 2.0
    * Visual Studio C# 2.0. ( need when building the MSI file) 
    * HTML Help Workshop ( needed when building the MSI file)
    * Doxygen 1.3.9.1 ( needed when building the MSI file)

How to build the setup msi file?

    * Load the IeUnit.sln into Visual Studio.

    * Switch to "Release" configuration.
 
    * Select "Rebuild solution" menu of Visual Studio.

    * The msi will be stored in the sub directory "build".
